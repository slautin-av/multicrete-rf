# DESIGN.md — переехал

Фирменная веб-дизайн-система МультиКрит теперь общая для всех проектов и живёт в «доме бренда»:

**➡ `D:\Project\multicrete-brand\DESIGN.md`** (рядом с `brand.md`)

Там — тёмная тема, токены (`:root`), компоненты, скевоморфизм, фон-дым, сетка, анимация, адаптивность и готовые шаблоны (включая шаблон кейса для сайта). Этот сайт (`multicrete-elastomers`, мультикрит.рф) — **эталонная реализация**: именно из его кода (`landing.html`, `calculator/calculator.css`) синхронизируется общий `DESIGN.md`.

Правило: правки визуального стиля вносим в код сайта, затем синхронизируем канонический `DESIGN.md` в `multicrete-brand`. Любой проект МультиКрит для артефакта в фирменном стиле обращается к канону (глобальное правило — в `~/.claude/CLAUDE.md`).

Источник истины бренда (логотип, палитра, шрифты, тон, юридика): `D:\Project\multicrete-brand\brand.md`.
