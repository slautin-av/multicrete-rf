# Входящие

Папка для сырых файлов на разбор. Алексей кладёт сюда документы, фото, выгрузки,
заметки — Claude разбирает: раскладывает по местам в проекте или удаляет черновики/мусор.

## Правила

- Claude может **перемещать и удалять файлы внутри этой папки без отдельного подтверждения** —
  это и есть «разгребание». Файлы за пределами «Входящих» — только с подтверждением Алексея.
- Перед удалением Claude коротко говорит, что именно убирает и почему.
- Полезное (калькуляции, рецептуры, кейсы, документы) — переносится в нужное место проекта
  с понятным именем (латиница-дефисы для кода; для документов-материалов допустима кириллица).
- Секреты/пароли/ключи из входящих файлов в код не попадают — выносятся в `.env`.

## Что сюда класть

- Калькуляции и Excel-выгрузки (для движка калькулятора)
- Фото деталей до/после + описания (будущие кейсы)
- Нормативные документы, прайсы, ГОСТы, заметки
- Всё, в чём нужно «разобраться и разложить»
