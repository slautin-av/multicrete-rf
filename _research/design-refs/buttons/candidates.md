# Кандидаты-кнопки для мультикрит.рф

Стилистика Алексея: **неоморфизм / скевоморфный минимализм** — тактильные кнопки в духе ретро-аудиотехники (Vega 109, Романтика). Современная версия физических кнопок старых усилителей.

## Финальный комплект для мультикрит.рф

Подбор завершён 2026-05-29. Двух типов кнопок достаточно для лендинга. Третий тип (icon со стрелкой/play) намеренно не брали — конкретной задачи под него нет.

### 1. Toggle (круглый переключатель)
- **Файл:** [toggle-pradeepsaranbishnoi.css](./toggle-pradeepsaranbishnoi.css)
- **Источник:** https://uiverse.io/Pradeepsaranbishnoi/jolly-robin-0
- **Стиль:** выпуклая круглая кнопка с фаской, мягкие тени, эффект «вжимания» при клике
- **Где применять:** переключатель режимов (м²/кг, RU/EN, ON/OFF слоёв покрытия), play в hero-видео

### 2. CTA (прямоугольная, два варианта)
- **Файл:** [cta-final-dual.md](./cta-final-dual.md)
- **Источники:**
  - Тёмный вариант (#212121): https://uiverse.io/minecrafte_8792/swift-pig-58
  - Светлый вариант (#e8e8e8): https://uiverse.io/javierBarroso/witty-wolverine-69
- **Стиль:** канонический неоморфизм — две противоположные тени дают выпуклость, при нажатии тени уходят внутрь (inset), кнопка реально «вжимается». Всего 20 строк CSS, легко кастомизируется.
- **Где применять:** «Запросить расчёт», «Скачать ТУ», «Связаться с инженером»

## Концепции для будущих экранов мультикрит.рф

Не отдельные кнопки, а **дизайн-паттерны** — применять там, где они логически просятся (калькулятор, селекторы, интерактивная инфографика).

### Панель переключателей в стиле приборной панели
- **Файл:** [концепция-панель-переключателей.md](./концепция-панель-переключателей.md)
- **Источник:** https://uiverse.io/m1her/splendid-rat-12
- **Суть концепции:** segmented control = «группа связанных переключателей». Тёмная подложка-«корпус» + ряд клавиш, торчащих наружу. Активная клавиша **вжимается** и **загорается голубым LED-свечением** — прямая имитация индикатора советского аудиоприбора (Vega, Романтика).
- **Почему это попадание:** говорит на языке физического оборудования. Главный инженер Газпрома узнаёт этот визуальный язык подсознательно — как панель управления, а не как сайт.
- **Где применять:**
  - Калькулятор покрытий: бетон / металл / дерево (выбор поверхности)
  - Селектор задачи: промышленный пол / антикор / гидроизоляция
  - Состав пакета: базовый / усиленный / экстрим
  - Переключатель слоёв покрытия в инфографике: грунт / основной / финиш
- **При вёрстке:** концепция важнее кода. Тёмная панель + вжимающаяся клавиша + LED-свечение активного — взять как принцип, реализацию можно адаптировать под палитру сайта (LED-цвет могут быть бирюзовый/жёлтый/зелёный по выбору).

Кнопки, которые понравились стилистически, но не для мультикрит.рф. Сохраняются для будущих проектов.

### Bluetooth-клавиша (Marcelo Dolza)
- **Файл:** [кандидат_1.md](./кандидат_1.md)
- **Источник:** https://uiverse.io/comported_2449/bad-snail-2
- **Автор:** Marcelo Dolza — [dolza.dev](https://dolza.dev/), [@cerpow](https://x.com/cerpow). Серьёзный референс по тактильным UI в целом.
- **Стиль:** гипер-скевоморфная клавиша с настоящим LED, переключение синий↔жёлтый, шумовая текстура. Самая тактильная кнопка из всего, что смотрели.
- **Почему не сейчас:** это toggle (фиксируется), а CTA уже закрыта простым неоморфизмом. Сайт мультикрит должен быть сдержаннее.
- **Когда пригодится:** интерактивные демо, играющие интерфейсы, проекты с сильной аудио/электронной эстетикой.
