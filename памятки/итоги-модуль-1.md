## Три ближайших шага

### 1. Создать главную страницу (`index.html`)
Шесть блоков-проблем крупно: ИЗНОС / ХИМЗАЩИТА / БЕТОН / МЕТАЛЛ / ЗАЩИТА / АВАРИЙНЫЙ РЕМОНТ.
Клиентский язык — боль, не материал. Каждый блок — иконка + слово + короткий подзаголовок → ссылка на страницу задачи.
Блок доверия: «производитель, не перекупщик» + цифры (объекты, лет работы).

### 2. Создать страницу одной задачи — для проверки шаблона
Взять самый понятный блок (например **ИЗНОС** или **АВАРИЙНЫЙ РЕМОНТ**).
Структура страницы: проблема → как мы её решаем → материал(ы) → кейс с цифрами → контактный призыв.
Если шаблон ок — тиражировать на остальные пять блоков.

### 3. Создать карточку одного материала — для проверки шаблона
МультиКрит 460 или МультиКрит 785 (у них уже есть аналог-конкурент → SEO-трафик).
Содержимое: характеристики, способ нанесения, сравнение с импортным аналогом, CTA «обсудить задачу», QR-код (заглушка).
