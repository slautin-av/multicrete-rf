# Скилл /kp — документы МультиКрит

## Хранилище документов (MK-Vault)

Все документы хранятся локально в `D:\MK-Vault\`. Notion не используется.

```
D:\MK-Vault\
├── шаблоны\                     ← бланки .docx
│   ├── БЛАНК_ПИСЬМА_МУЛЬТИКРИТ.docx     (КП и письма)
│   ├── БЛАНК_ДОГОВОРА_ПОДРЯД.docx
│   └── БЛАНК_ДОГОВОРА_ПОСТАВКА.docx
├── документы\КП\
├── документы\договоры\
├── документы\письма\
├── журналы\журнал-КП.md
├── журналы\журнал-договоров.md
└── журналы\журнал-писем.md
```

Перед присвоением номера КП или договора — читать соответствующий журнал, чтобы не дублировать.
После создания документа — обновлять журнал (дописывать строку в таблицу).

## Режимы работы

### РЕЖИМ 1: КОММЕРЧЕСКОЕ ПРЕДЛОЖЕНИЕ

**Нумерация:** КП-001, КП-002 ... (проверить последний в журнал-КП.md)

**Опросный лист для работ:**
1. Наименование Заказчика (полное и краткое)
2. Объект работ (что покрываем/ремонтируем)
3. Место выполнения — Ижевск или выезд?
4. Калькуляция загружена? (если нет — площадь, толщина, материал)
5. Условия оплаты (100% предоплата / 70%+30% / иное)
6. Срок выполнения работ
7. Гарантийный срок (по умолчанию 12 мес.)
8. Кто выполняет дробеструйную подготовку?
9. Доставка оборудования — кто, за чей счёт?

**Опросный лист для поставки:**
1. Наименование Заказчика
2. Перечень продукции (наименование, кол-во, цена)
3. Условия поставки (самовывоз / доставка)
4. Условия оплаты
5. Срок поставки (по умолчанию: 20 рабочих дней с оплаты)
6. Срок годности товара (по умолчанию: 36 мес. на складе Покупателя)

**Подбор материала по задаче:**
- ИЗНОС (абразив, кавитация, трение) → Серия 3 (КерамоКомпозиты), Серия 7 (Эластомеры 700)
- ХИМЗАЩИТА (кислоты, щёлочи) → Серия 4 (МСР 460), Серия 5
- ВОССТАНОВЛЕНИЕ БЕТОНА → Серия 1 (КомпозитБетоны)
- РЕМОНТ МЕТАЛЛА / ХОЛОДНАЯ СВАРКА → Серия 2 (МеталлоКомпозиты)
- АВАРИЙНОЕ УСТРАНЕНИЕ ТЕЧИ → Серия 2, Rapid (В=6)
- УДАРНЫЕ НАГРУЗКИ / ШУМ / НАЛИПАНИЕ → Серия 7 (Эластомеры 700)

**Структура КП:**
```
КОММЕРЧЕСКОЕ ПРЕДЛОЖЕНИЕ № КП-XXX
МультиКрит | г. Ижевск | multicrite.ru
Дата: ДД.ММ.ГГГГ

Кому: [Заказчик], [Контактное лицо]

Ваша задача:
[Описание проблемы клиента — 2-3 предложения]

Наше решение:
[Материал МультиКрит + почему он подходит]

Таблица позиций:
| № | Наименование | Ед. | Кол-во | Цена | Сумма |

Итого: X руб. НДС не облагается (УСН)

Условия: оплата, срок, гарантия, доставка
Срок действия КП: 30 дней

Алексей Слаутин, учредитель МультиКрит
```

**Спросить после составления КП:**
- «Простое КП (таблица + условия) или развёрнутое (с анализом и экспертизой)?»
- «Утверждаем? Готовить файл .docx?»

### РЕЖИМ 2: ДОГОВОР

**Нумерация:**
- Подряд (работы): ВР-01-04/26 (порядковый-месяц/год)
- Поставка: ПМ-27-04/26 (день-месяц/год)

**Тип договора:**
- Есть работы по нанесению/ремонту → договор подряда (бланк БЛАНК_ДОГОВОРА_ПОДРЯД.docx)
- Только материалы/продукция → договор поставки (бланк БЛАНК_ДОГОВОРА_ПОСТАВКА.docx)

**Принципы формулировок (МультиКрит — Исполнитель/Поставщик):**
- Стоимость ТВЁРДАЯ, изменению не подлежит
- Подрядчик ВПРАВЕ привлекать субподрядчиков (уведомительно)
- Подрядчик ВПРАВЕ приостановить работы при неоплате
- Молчание заказчика 5 дней = приёмка
- Гарантия не распространяется на нарушение правил эксплуатации, ремонт третьими лицами

**Чеклист перед выдачей договора:**
- [ ] ИНН 1800008000, КПП 180001001, ОГРН 1231800018394 — верно?
- [ ] Банк ВТБ, р/с 40702810915900002419, БИК 044525411 — верно?
- [ ] НДС не облагается (УСН) — указано?
- [ ] Сумма договора = сумма в КП/калькуляции?
- [ ] Подсудность — АС Удмуртской Республики?
- [ ] НЕТ банковских гарантий?
- [ ] НЕТ гарантийного удержания?
- [ ] Ответственность ограничена суммой договора?
- [ ] Пеня подрядчика ≤ 10% от стоимости работ?
- [ ] Номера приложений совпадают с реальными приложениями?

### РЕЖИМ 3: АНАЛИЗ ДОГОВОРА ЗАКАЗЧИКА

1. Прочитать договор полностью
2. Выдать таблицу ключевых условий: параметр / значение / наша позиция / статус (✅⚠️❌)
3. Выявить подводные камни: подсудность, банковские гарантии, гарантийные удержания, несимметричные штрафы, требования СРО, строительные нормы для покрытий
4. Предложить: Вариант А (правки в их договор) или Вариант Б (протокол разногласий)

### РЕЖИМ 4: ПИСЬМА

Типы: КП / Деловое письмо / Претензия / Сопроводительное / Ответ на запрос
Нумерация: Исх. № КП-001 от ДД.ММ.ГГГГ (проверить последний в журнал-писем.md)

## Алгоритм вывода любого документа

1. Показать полный текст в чате (markdown)
2. Спросить: «Всё верно? Вносим правки или готовим .docx?»
3. После явного подтверждения — записать файл в `D:\MK-Vault\документы\{тип}\`
4. Дописать строку в соответствующий журнал в `D:\MK-Vault\журналы\`

## Важно

- Не выдумывать технические характеристики — только достоверные факты
- Если задача нестандартная — писать «требует технологической консультации»
- Договор НЕ оформляется на фирменном бланке — только КП и письма
- Тон: профессиональный, без канцелярита, партнёрский
