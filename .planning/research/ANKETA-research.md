# Исследование: универсальный опросный лист для подбора эластомерного покрытия

*Составлено 2026-07-01 по результатам deep-research (22 источника, 84 факта → 25 проверено → 17 подтверждено тройным голосованием). Источники — сайты и техдокументация прямых аналогов МультиКрит.*

## Зачем это исследование
Проверить, как ведущие производители износостойких/защитных покрытий собирают данные о детали заказчика, и не изобретать велосипед при создании ЕДИНОГО универсального опросного листа МультиКрит (серии 700–800).

---

## 1. Главный инженерный вывод — от чего вообще зависит, подойдёт ли эластомер

**Угол атаки абразива/потока — решающий критерий пригодности эластомера** (подтверждено, Belzona + academic):
- **Малый угол (< 30°), скользящее истирание** (прямые участки труб, стояки) — «режущий» (ductile) износ. Здесь эластомер изнашивается быстрее, **лучше работают твёрдые/хрупкие материалы** (композиты, керамика, твёрдый металл).
- **Большой угол (60–90°), удар по нормали** (колена, отводы, крыльчатки, зоны кавитации) — «ударный» (brittle) износ. Здесь **эластомер выигрывает** — резиноподобная упругость поглощает энергию удара; твёрдые хрупкие покрытия трескаются.

→ **Практический смысл для листа:** нужно спрашивать характер воздействия (истирание вскользь vs удар/кавитация). Это сразу говорит, наш ли это продукт (эластомер) или клиенту нужен композит/керамика. Прямой аналог MetaLine так и делит: **Series 700 (эластомер) — против эрозии, кавитации, ударов и вибрации; НЕ подходит для трения и статических линейных нагрузок** — там рекомендуют керамическую серию.

**Твёрдость по Шор А — реальный, но неоднозначный критерий** (подтверждено):
- Под **эрозию** — мягче лучше (мягкий ПУ поглощает удары частиц).
- Под **абразивное истирание** — твёрже лучше (сопротивляется вдавливанию частиц).
- → Оптимум зависит от механизма износа. У МультиКрит твёрдость уже кодируется (Г-код: 45/55/65/75/85/95 Шор А), аналог MetaLine: 785 = 85А, 795 = 95А. Значит лист должен собрать данные, по которым выбирается твёрдость.

**Абразив: размер + угловатость + твёрдость по Моосу + концентрация** (подтверждено):
- Угловатые частицы тверже Мооса 5 — эластомер/резина не спасёт, нужен твёрдый металл; округлые и мягче — годится эластомерная футеровка.
- → Лист должен спрашивать не только «есть абразив», но фракцию, форму, твёрдость и концентрацию.

**Температура — жёсткая граница, отсекающая марку** (подтверждено):
- Пример Belzona: 1331 годен до +50 °C в погружении, 1381 — до +95 °C. Марка выбирается по температуре.
- → Собирать рабочую температуру среды: **постоянную И пиковую**, отдельно для погружения.

**Химсреда — концентрация + температура + длительность контакта** (подтверждено/частично):
- Chesterton кодирует стойкость не как «да/нет», а по 4 уровням экспозиции (постоянно / кратковременно <72 ч / проливы с уборкой <8 ч / нельзя) и отдельно для разных концентраций и температур одного и того же вещества.
- → Лист должен спрашивать вещество + концентрацию + температуру + характер контакта (постоянно/периодически/проливы).

**Кавитация — отдельный критерий** (подтверждено): решается упругостью + адгезией, тестируется по ASTM G32. Важна для крыльчаток и гребных винтов. Твёрдые хрупкие покрытия при углах > 60° не годятся.

**Подготовка поверхности и адгезия к основе — предусловие работоспособности** (подтверждено): в ARC прямо нормируются степень очистки (Sa 2.5 / SP11) и профиль анкеровки. → В листе нужен материал основы + её состояние (новая/изношенная, остаточная толщина стенки, коррозия).

## 2. Что собирают конкуренты (перечни полей)

**Матрица подбора Chesterton ARC** — оси, по которым классифицируются покрытия (значит, эти же поля нужны на входе):
- Основа: металл / бетон
- Температура погружения по диапазонам: <60 / <90 / <130 / <150 / <180 °C
- Химсреда: конц. кислота / разб. кислота / щёлочь / питьевая вода
- Эрозия: слабая / сильная
- Абразия: умеренная / сильная / экстремальная
- Удар: слабо-умеренный / умеренно-сильный
- + профиль поверхности и степень очистки

**MetaLine (прямой аналог, эластомеры Series 700):** сегментирует подбор по механизму разрушения (эрозия / кавитация / удар / вибрация) и по типу детали (крыльчатки, насосы). Ключевой параметр — твёрдость (785/795 Шор А) и толщина слоя.

**Belzona (насосы):** ядро критериев — химстойкость + высокотемпературная стойкость + эрозионная + кавитационная стойкость; механизмы: эрозия, коррозия, химатака, износ, кавитация.

**ADL (российский опросник подбора насоса, как образец структуры листа)** — частично подтверждено:
- Блок «тип применения» (водоснабжение/отопление/дренаж/канализация/другое) как рамка для остальных полей.
- Состав среды: тип жидкости + концентрация + % примесей + макс. размер частиц.
- Контактный блок (организация / контактное лицо / e-mail / телефон / город) — отдельно от технического.
- Предлагается И онлайн-форма, И скачиваемый DOC (модель «двойной доставки» — но нам онлайн не нужен по решению владельца).

**Российские гайды по опросным листам:** опросный лист = стандартизованное ТЗ; чем точнее заполнен — тем быстрее КП. Типовые блоки: базовые данные (наименование/назначение в техпроцессе) → тех. параметры (давление, расход, температура, мощность) → условия эксплуатации → требования → материалы/интерфейсы.

## 3. Вывод по универсальному листу
Единый лист — правильно и подтверждается практикой (Chesterton/Belzona/MetaLine используют ОДНУ матрицу критериев на все отрасли; отрасль — лишь фильтр). Структура «объект → механизм разрушения» валидна: одни и те же детали (трубы, насосы, клапаны, циклоны) в литературе идут единым списком именно по механизму износа, а не по отрасли. Это совпадает с философией сайта МультиКрит («по типу проблемы, не по отрасли»).

**Что из этого добавить к бланку по насосам (чего там не хватает):**
1. **Характер воздействия / угол атаки** — истирание вскользь vs удар по нормали / кавитация (решает, эластомер ли вообще нужен).
2. **Форма и твёрдость абразива** (угловатость, Моос), не только фракция и концентрация.
3. **Температура — постоянная И пиковая**, отдельно для погружения.
4. **Химия — концентрация + температура + характер контакта** (постоянно/периодически/проливы), а не только вещество+%.
5. **Состояние основы** — новая/изношенная, остаточная толщина, наличие коррозии (для адгезии/подготовки).
6. **Требуемый ресурс / межремонтный интервал** и **габарит доступа** (можно ли снять деталь, толщина слоя).
7. **Селектор применения** (насос / винт / труба / циклон / мельница / вентилятор / ёмкость / арматура / другое) как рамка — неприменимые поля заказчик помечает прочерком.

## Оговорки по достоверности
- Убито голосованием (НЕ использовать как факт): «эластомер не годится при постоянной t среды > 60 °C» (0-3 — слишком категорично); «твёрдость наполнителя должна превышать твёрдость абразива» (0-3, спорно для эластомеров).
- Не добито из-за лимита сессии (направление верное, но проверить при планировании): 4-уровневая шкала химэкспозиции Chesterton; двойная доставка ADL (онлайн+DOC); стартовый селектор применения ADL.
