# Requirements: Портал мультикрит.рф — Публичный калькулятор покрытий

**Defined:** 2026-06-29
**Core Value:** Калькулятор достоверно прогоняет метод расчёта Алексея (классы сложности × коэффициенты × материал/толщина) и выдаёт предварительную стоимость с прозрачной разбивкой — ориентир, которому можно доверять.

## v1 Requirements

Требования для первого выпуска публичного калькулятора. Каждое ложится в этап дорожной карты.

### Движок расчёта (CALC)

- [ ] **CALC-01**: Движок воспроизводит логику утверждённого Excel (`Калькуляция МультиКрит 700.xlsx`) для входов площадь + толщина + класс — итоговые цифры совпадают с эталоном в пределах округления
- [ ] **CALC-02**: Класс сложности 1/2/3 применяет множитель ×1.0 / ×1.2 / ×1.4 к трудозатратам (к работам, не к материалам)
- [ ] **CALC-03**: Стоимость нанесения считается по толщине нелинейно (первые 3 мм по повышенной расценке, далее — по сниженной)
- [ ] **CALC-04**: Расход и стоимость материала считаются по площади и толщине с учётом ценовых порогов по массе эластомера
- [ ] **CALC-05**: Вспомогательные материалы считаются по нормативам от площади (исправленная формула: стоимость = количество × цена, итог = сумма)
- [ ] **CALC-06**: Накрутки применяются в правильном порядке: ЕСН на труд, накладные, склад, гарантийный резерв, плановые накопления (прибыль); налог УСН учтён, НДС = 0
- [ ] **CALC-07**: Грузоподъёмное оборудование подключается отдельной строкой при флаге «деталь тяжелее 80 кг»
- [ ] **CALC-08**: Фиксированные допущения зашиты по умолчанию: 1 день работ, 1 специалист, без транспорта, без командировок, без спецоснастки
- [ ] **CALC-09**: Все коэффициенты, ставки и цены вынесены в единый блок параметров внутри кода (один источник истины, не разбросаны по формулам)

### Интерфейс калькулятора (UI)

- [ ] **UI-01**: Калькулятор открывается всплывающим окном поверх страницы сайта и закрывается обратно
- [ ] **UI-02**: Поля ввода: площадь детали (м²), толщина покрытия (мм), класс сложности (1/2/3), галочка «деталь тяжелее 80 кг»
- [ ] **UI-03**: Результат показывается «шапкой» сметы: общая стоимость работ, в т.ч. трудозатраты, вспомогательные материалы, накладные расходы, плановые накопления, налоги, командировочные = 0, стоимость материалов, ИТОГО и ₽/м²
- [ ] **UI-04**: Рядом с калькулятором — расшифровка классов 1/2/3 (примеры деталей) и блок оговорок (1 день, 1 специалист, без транспорта/командировок/оснастки, грузоподъёмное только при >80 кг)
- [ ] **UI-05**: Ввод проверяется на корректность (положительные числа, выбран класс); при ошибке — понятная подсказка
- [ ] **UI-06**: Калькулятор выполнен в фирменном стиле сайта и корректно работает на телефоне (адаптивность)

### Встраивание в сайт (INT)

- [ ] **INT-01**: Калькулятор живёт как отдельная страница/модуль `calculator/` по аналогии с `materials/`, на чистом HTML/CSS/JS без сборщика
- [ ] **INT-02**: На лендинге есть заметная кнопка-вызов калькулятора

### Заявка / обратная связь (LEAD)

- [ ] **LEAD-01**: После результата расчёта доступна НЕОБЯЗАТЕЛЬНАЯ форма заявки: имя, организация, телефон, e-mail (цена видна без заполнения формы)
- [ ] **LEAD-02**: Форма отправляет заявку владельцу через сторонний сервис-приёмник (без своего сервера); при успехе — подтверждение, при ошибке — понятное сообщение
- [ ] **LEAD-03**: Рядом с формой — галочка согласия на обработку персональных данных (152-ФЗ) со ссылкой на политику; без согласия отправка недоступна

## v2 Requirements

Отложено на будущие выпуски — учтено, но не в текущей дорожной карте.

### Опросник для подбора покрытия (ANKETA)

- **ANKETA-01**: Опросник-анкета для клиентов, которые не знают, какое покрытие нужно: тип детали, среда (химия/абразив), соотношение твёрдых частиц к воде, сухая/мокрая, температура, фракция абразива, кавитация, налипание, обороты вращения
- **ANKETA-02**: По ответам МультиКрит подбирает покрытие (логика подбора — инженерная экспертиза владельца)
- **ANKETA-03**: Анкета доступна на сайте отдельным блоком и/или скачиваемым бланком для отправки на e-mail
- _Основа: существующий бумажный бланк-опросник владельца (разобрать и сохранить в проект)_

### Расширение расчёта (CALC2)

- **CALC2-01**: Посегментный расчёт нанесения вместо средней толщины
- **CALC2-02**: Минимальный чек / стоимость выезда для мелких деталей
- **CALC2-03**: Полный режим со сметой по сегментам, флагами (командировки, дилер, оснастка) — перенос всего Excel на сайт

### Лидогенерация — расширение (LEAD2)

- **LEAD2-01**: Печать/сохранение предварительного расчёта в PDF
- **LEAD2-02**: Отправка заявок дополнительно в Telegram (после запуска Telegram-бота)

## Out of Scope

Явно исключено, чтобы не вернуть случайно.

| Feature | Reason |
|---------|--------|
| Серверный слой, сокрытие коэффициентов | Владелец работает прозрачно; расчёт идёт в браузере |
| Посегментная геометрия и ручные флаги на сайте | Это уже есть в Excel; на сайте — упрощённый расчёт по 3 входам |
| Автокейсы, каталог проектов | Отдельная следующая большая цель |
| Telegram-бот | Отдельный проект в очереди |
| Переделка бренда (логотип/палитра/шрифты) | Живёт в соседнем проекте `multicrete-brand` |
