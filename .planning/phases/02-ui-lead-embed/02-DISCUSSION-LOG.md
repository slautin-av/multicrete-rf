## Куда уходит заявка

| Вариант | Описание | Выбран |
|--------|-------------|----------|
| Web3Forms | Зарубежный приёмник по ключу, без аккаунта, форма остаётся нашей | ✓ (как fallback) |
| Formspree | Похоже, но нужен аккаунт, ~50 заявок/мес бесплатно | |
| Российский путь (152-ФЗ) | Строго по локализации, но больше настройки / чужой виджет | |
| PHP на собственном хостинге Reg.ru | Заявка → скрипт на своём хостинге → почта владельца | ✓ (основной) |

**User's choice:** Сначала выбрал Web3Forms; затем выяснилось, что у владельца уже есть оплаченный хостинг
Reg.ru (поддерживает PHP/почту). Перерешили: основной путь — PHP-обработчик на Reg.ru (данные в РФ, по
152-ФЗ, бесплатно), Web3Forms оставлен запасным вариантом.

**Notes:** Владелец прояснил путаницу «сервер vs хостинг»: хостинг (место для файлов) есть, сервер-обработчик
можно сделать на нём же через PHP. Доступы пришлёт позже. Охват: в этом этапе обработчик — только для формы
калькулятора. Две формы лендинга («звонок», «прайс») — отложены; блок «представители в регионах» — отложен.

## Как открыть калькулятор

| Вариант | Описание | Выбран |
|--------|-------------|----------|
| Окно поверх лендинга (оверлей) | Всплывающее окно, клиент не теряет контекст, переиспользуем готовую механику | ✓ |
| Отдельная страница | Кнопка ведёт на страницу calculator/, больше места, но уход с лендинга | |

**User's choice:** Окно поверх лендинга.
**Notes:** Калькулятор в любом случае собирается как модуль `calculator/`; прямую ссылку-страницу можно дать позже.

## Глубина сметы в итоге

| Вариант | Описание | Выбран |
|--------|-------------|----------|
| «Шапка» + разворот | Сводка по умолчанию, «Показать полный расчёт» разворачивает построчно | частично |
| Только «шапка» | Сгруппированная сводка без построчного низа | |
| «Шапка» бесплатно + детализация за контакты | ИТОГО видно сразу; полная раскладка — после ввода контактов | ✓ |

**User's choice:** «Шапка» и ИТОГО бесплатно; полная построчная детализация раскрывается после заполнения
контактов (имя, телефон, e-mail) — собственная идея владельца (воронка заявок).
**Notes:** Цена видна без формы (LEAD-01 соблюдено), форма остаётся необязательной. Оговорки и расшифровка
классов показываются сразу с «шапкой» — не прячем за форму.
