# Phase 2 Plan 03: Страница политики 152-ФЗ — Summary

Создана отдельная статическая страница `privacy/index.html` — политика обработки персональных данных по типовому шаблону 152-ФЗ в тёмной теме сайта; реквизиты оператора оставлены плейсхолдерами под подстановку владельцем. На неё будет ссылаться чекбокс согласия формы калькулятора (план 04).

## Что сделано

- **Задача 1 — `privacy/index.html`** (коммит `0e14abf`)
  - 8 обязательных разделов 152-ФЗ: (1) Общие положения, (2) Термины и определения, (3) Состав обрабатываемых данных, (4) Цели обработки, (5) Правовые основания, (6) Порядок и условия обработки, (7) Права субъекта ПДн, (8) Защита данных и контакты оператора.
  - Тёмная тема «Стальной Vega»: инлайн-`<style>` с `:root` (--blue/--ink/--steel/--scale/--line/--muted/--bg/--f-head/--f-body/--f-tech), скопированными из `landing.html`. Светлая палитра сайта не подключалась (D-08).
  - Шрифты бренда (Oswald/Manrope/Tektur) через Google Fonts, как в `landing.html`.
  - Реквизиты оператора — плейсхолдерами: `{{НАИМЕНОВАНИЕ}}`, `{{ИНН}}`, `{{ОГРН}}`, `{{ЮР_АДРЕС}}`, `{{ДАТА}}`. Визуально выделены (класс `.ph`) для удобства замены владельцем.
  - Контакты оператора (раздел 8) подставлены сразу из `landing.html`: телефоны +7 (3412) 32-41-51 / +7 (965) 842-41-51, e-mail MultiCrete@yandex.ru, адрес «г. Ижевск, ул. 40 лет Победы, д. 92».
  - Указано: хостинг РФ, трансграничной передачи нет, передача третьим лицам — только по закону/с согласия.
  - Страница доступна по пути `/privacy/` (index.html в папке privacy/), есть ссылка «Назад на сайт».
  - Адаптив: `clamp()` для заголовка, медиазапрос ≤560px.
