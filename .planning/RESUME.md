## Ключевые решения владельца (2026-06-29)
- **Калькулятор публичный и прозрачный.** Алексей сам показывает клиентам полную калькуляцию — скрывать нечего. Считаем прямо в браузере, коэффициенты НЕ прячем. (Прежняя установка «коэффициенты не в браузер» ОТМЕНЕНА.)
- **На сайт — упрощённый расчёт по 3 входам:** площадь + толщина + класс сложности (1/2/3) + флаг «деталь >80 кг». Полная посегментная смета остаётся в Excel.
- **Показываем конкретные цифры** (итоговую «шапку» сметы), не размытую вилку.
- **Форма заявки** после результата — необязательная (имя, организация, телефон, e-mail), отправка через сторонний сервис-приёмник (без своего сервера), с галочкой согласия 152-ФЗ. Цена видна без заполнения формы.
- **Опросник-анкета** (подбор покрытия по среде/абразиву/температуре/кавитации/оборотам) — отдельная следующая цель (v2 ANKETA), НЕ в этом милстоуне. Есть бумажный бланк владельца → разобрать и сохранить (задача в STATE.md).

## Модель в двух словах
Считает не площадь, а СЛОЖНОСТЬ детали. Цепочка: геометрия → расход материала (с потерями) → построчная смета работ → **× коэффициент класса** (1=×1.0 / 2=×1.2 / 3=×1.4) → ИТР → взносы 30.2% → накладные 43% → склад 1.5% → гарантия 3% → плановые накопления 60% → ИТОГО. Налоги: УСН, НДС=0. Материал — по прайсу без наценки (производят сами). Для сайта фиксируем допущения: 1 день, 1 специалист, без командировок/транспорта/оснастки; грузоподъёмное — только при >80 кг.
