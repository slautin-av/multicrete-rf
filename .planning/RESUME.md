## Модель в двух словах
Считает не площадь, а СЛОЖНОСТЬ детали. Цепочка: геометрия сегментов → расход материала (с потерями) → построчная смета работ → **× коэффициент класса** (A=1.0 простое тело вращения / B=1.2 сложное-импеллер / C=1.4 с маскировкой) → ИТР → взносы 30.2% → накладные 43% → склад 1.5% → прибыль 60% → гарантия 3% → ИТОГО. Налоги: УСН, НДС=0. Материал — по прайсу без наценки (производят сами). Дробеструйка — переключатель «выполняет заказчик» (0) / «выполняем мы».

## Два режима веб-калькулятора (из ВИДЕНИЕ-ПОРТАЛА.md)
- **Заказчику** — упрощённый: тип детали + габариты → ВИЛКА ±20% и ₽/м², без разбивки.
- **Алексею** — полный: точная смета (как Excel).
- 🔒 ПРИВАТНОСТЬ: коэффициенты/прайс/маржа — НЕ в браузер. Только на сервере/в `.env`. Публично — лишь вилка. Это жёсткое бизнес-правило (House-фильтр).
