# Roadmap: Публичный калькулятор покрытий мультикрит.рф

## Overview

Переносим утверждённый Excel-калькулятор стоимости полимерного покрытия в публичный калькулятор на сайте. Путь: сначала собираем и проверяем сам движок расчёта на чистом JavaScript (его цифры должны совпасть с эталонным Excel), затем надеваем поверх него понятный интерфейс — всплывающее окно с вводом трёх параметров, итоговой «шапкой» сметы, оговорками и расшифровкой классов — и встраиваем на сайт в фирменном стиле. Весь расчёт идёт в браузере: владелец работает прозрачно, прятать коэффициенты не нужно.

## Phase Details

### Phase 1: Движок расчёта
**Goal**: Готовый проверяемый движок на JavaScript, который по входам (площадь детали, толщина покрытия, класс сложности 1/2/3) выдаёт полную разбивку предварительной стоимости, совпадающую с утверждённым Excel-калькулятором; в конце расчёта — текстовая оговорка, что деталь бралась массой до 80 кг.
**Depends on**: Nothing (first phase)
**Requirements**: CALC-01, CALC-02, CALC-03, CALC-04, CALC-05, CALC-06, CALC-07, CALC-08, CALC-09
**Success Criteria** (что должно быть ПРАВДОЙ):
  1. На контрольных примерах (минимум 2-3) движок возвращает ИТОГО и ₽/м², совпадающие с эталонным Excel в пределах округления
  2. Переключение класса 1 → 2 → 3 умножает трудозатраты ровно на ×1.0 / ×1.2 / ×1.4
  3. Результат содержит текстовую оговорку, что расчёт сделан для детали массой до 80 кг и при большей массе добавится грузоподъёмное оборудование (стоимость вырастет); отдельная расчётная строка не считается
  4. Стоимость нанесения меняется нелинейно по толщине (первые 3 мм по повышенной расценке, далее — по сниженной)
  5. Все коэффициенты, ставки и цены собраны в одном блоке параметров (один источник истины)
**Plans**: 2 plans
  - [x] 01-01-PLAN.md — единый блок параметров PARAMS (один источник истины, CALC-09)
  - [x] 01-02-PLAN.md — ядро calc() + текстовая оговорка + самопроверка на контрольных примерах эталона

### Phase 2: Интерфейс, заявка и встраивание
**Goal**: Клиент на сайте открывает калькулятор всплывающим окном, вводит площадь/толщину/класс (и при необходимости флаг «>80 кг»), сразу видит предварительную стоимость с разбивкой-«шапкой», оговорками и расшифровкой классов, а затем при желании оставляет заявку (контакты) — в фирменном стиле и на телефоне.
**Depends on**: Phase 1
**Requirements**: UI-01, UI-02, UI-03, UI-04, UI-05, UI-06, INT-01, INT-02, LEAD-01, LEAD-02, LEAD-03
**Success Criteria** (что должно быть ПРАВДОЙ):
  1. С лендинга заметной кнопкой открывается всплывающее окно калькулятора и закрывается обратно
  2. После ввода площадь/толщина/класс (+флаг) показывается «шапка» сметы: общая стоимость работ, в т.ч. трудозатраты, вспомогательные материалы, накладные, плановые накопления, налоги, командировочные = 0, стоимость материалов, ИТОГО и ₽/м²
  3. Рядом с расчётом видны расшифровка классов 1/2/3 и блок оговорок (1 день, 1 специалист, без транспорта/командировок/оснастки, грузоподъёмное только при >80 кг)
  4. Цена видна без заполнения контактов; после результата доступна необязательная форма заявки (имя, организация, телефон, e-mail) с галочкой согласия на обработку перс. данных, которая отправляет заявку владельцу и показывает подтверждение
  5. Некорректный ввод даёт понятную подсказку и не ломает расчёт; калькулятор выполнен в фирменном стиле и корректно отображается на телефоне
**Plans**: 5 plans

  - [x] 02-02-PLAN.md — PHP-обработчик заявки send-lead.php (защита от инъекций, honeypot, mail на владельца) (LEAD-02) [волна 1]
  - [x] 02-03-PLAN.md — страница политики 152-ФЗ privacy/index.html (8 разделов, плейсхолдеры реквизитов) (LEAD-03) [волна 1]
  - [x] 02-04-PLAN.md — форма заявки + согласие 152-ФЗ + отправка + воронка детализации (LEAD-01, LEAD-02, LEAD-03) [волна 2]
  - [ ] 02-05-PLAN.md — встраивание окна в лендинг + кнопка «Получить расчёт» + ручной чек-лист приёмки (INT-02) [волна 3]
