# MultiCrete.ru

B2B-сайт МультиКрит — промышленные полимерные покрытия.

## Структура проекта

```
multicrite-ru/
├── index.html              ← Главная страница (5 блоков проблем)
├── problems/               ← Страницы по типу проблемы
│   ├── iznos/              ← ИЗНОС
│   ├── koroziya/           ← КОРРОЗИЯ
│   ├── himiya/             ← ХИМИЯ
│   ├── remont/             ← РЕМОНТ
│   └── germetizaciya/      ← ГЕРМЕТИЗАЦИЯ
├── materials/              ← Карточки материалов + аналоги импортных
├── cases/                  ← Кейсы с реальными цифрами
├── knowledge/              ← База знаний
│   ├── videos/             ← Синхронизация с YouTube
│   └── articles/           ← Синхронизация с ВК/Telegram
├── about/                  ← О компании
└── assets/                 ← Общие ресурсы
    ├── images/             ← Картинки, логотип
    └── styles/             ← Общие CSS-стили
```

## Концепция навигации

Навигация — по типу проблемы, не по материалу и не по отрасли.
Клиент не знает что его проблему решают полимерные материалы — он знает свою проблему.
